文章目录

  2. 2. 本地化图片api

本篇文章主要讲的是简单制作图片/视频API的一个方法，做属于自己的API（大神绕道）
适合新手，适合懒人，

外链版图片API

首先需要外链图片（一行一个链接）缺点就是有时候图片会重复

演示：https://api.ly522.com/

制作方法

新建一个PHP文件（名字自己取）在创建一个images.txt文档文件，把外链图片地址放进去

上传服务器空间 然后访问域名/PHP文件名就可以啦

代码放在压缩文件了，下方有下载（我把代码高亮给搞异常了所以就是这样）
